KiCad Beginner Cheat Sheet

Save this somewhere.

LEDs
Classic Arduino LED (THT)

The ones from Arduino kits.

LED_THT:LED_D5.0mm

Looks like:

 O
/ \
Small LED
LED_THT:LED_D3.0mm

Smaller version.

SMD LEDs

Common beginner SMD sizes:

LED_SMD:LED_0805_2012Metric
LED_SMD:LED_0603_1608Metric

Start with:

LED_0805

because it's easier to see.

Resistors
Arduino Kit Resistor
Resistor_THT:R_Axial_DIN0207_L6.3mm_D2.5mm_P7.62mm_Horizontal

The classic one you've already used.

Vertical Resistor

Takes less board space.

Resistor_THT:R_Axial_DIN0207_P2.54mm_Vertical
SMD Resistors

Large beginner-friendly:

Resistor_SMD:R_0805_2012Metric

Smaller:

Resistor_SMD:R_0603_1608Metric

Tiny:

Resistor_SMD:R_0402_1005Metric

Avoid 0402 for now.

Use:

0805
